{%- comment -%} _includes/json-ld.html Emits a schema.org JSON-LD block for the current page with full author and license metadata. This is the machine-readable attribution signal that AI systems, citation tools, and search engines consume. Author and license values come from _config.yml (content_license, first_name, last_name, email, url). Per-page overrides via front matter: - license: "CC BY-SA 4.0" # override default - author: "Different Person" # override default author - author_url: "https://..." # override default author URL To include in your theme, add this line near the top of _includes/head.html: {%- include json-ld.html -%} {%- endcomment -%} {%- assign page_type = "CreativeWork" -%} {%- if page.layout == "post" -%} {%- assign page_type = "ScholarlyArticle" -%} {%- endif -%} {%- assign author_name = page.author | default: site.social.name -%} {%- if author_name == nil or author_name == "" -%} {%- assign author_name = site.first_name | append: " " | append: site.last_name -%} {%- endif -%} {%- assign license_url = page.license_url | default: site.content_license.url -%} {%- assign license_name = page.license | default: site.content_license.short -%} {%- comment -%} Also emit the standard for crawlers that read HTML head links rather than JSON-LD. {%- endcomment -%}
