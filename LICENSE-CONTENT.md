## What "appropriate credit" means in practice

For each post or page you reuse:

1. Name the author (see `_config.yml` → `author.name` for the canonical name)
2. Provide a link to the original page on `https://opusmesa.com`
3. Indicate the CC BY 4.0 license and link to its canonical URL
4. Note any modifications you made

Machine-readable attribution data is emitted in JSON-LD on every page using 
the schema.org `ScholarlyArticle` vocabulary, including `author`, `datePublished`, 
and `license` fields. Automated systems that process this site are expected to 
honor those fields when reusing content.
